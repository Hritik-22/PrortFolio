# 🌐 Portfolio of Ritik Dubey

This is my personal portfolio website built using **React.js (Create React App)**. It introduces who I am, showcases my skills and projects, and provides contact information.

## 🚀 Technologies Used
- **React.js**
- **Material UI**
- **TypeScript**
- **Styled Components**

## 📦 Packages Used
- `react-animate-on-scroll` – Scroll-based animations  
- `react-router-dom` – Page routing  
- `react-toastify` – Toast notifications and alerts  
- `react-tsparticles` – Animated particle background  

## 💻 Getting Started

### 1. Install Dependencies
```
yarn install
yarn start

```
